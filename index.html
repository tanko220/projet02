How Internet works

How does the Internet work? Good question! The growth of the Internet has become explosive and it seems impossible to escape the bombardment of www.com constantly seen on television, heard on the radio and seen in magazines. Because the Internet has become such an important part of our lives, a good understanding is needed to use this new tool most effectively.

Internet Protocols

Internet Protocol (IP) is the network layer communication protocol of the Internet protocol suite for relaying datagrams across network boundaries.

IP Addresses

An IP address is a unique address that identifies a device on the Internet or a local network. ... Essentially, IP addresses are the identifier that allows information to be sent between devices on a network: they contain location information and make devices accessible for communication.

DNS

DNS (Domain Name System) is like the Internet phone book. Internet users access information online through domain names (for example, nytimes.com or espn.com), while browsers interact through Internet Protocol (IP) addresses. DNS translates domain names into IP addresses so that browsers can load web resources.

Packets TCP/UDP

TCP (Transport Control Protocol) provides reliable data transmission service with end-to-end error detection and correction. UDP (User Datagram Protocol) provides connectionless datagram transmission service.

HTTP

Hypertext Transfer Protocol is a client-server communication protocol developed for the World Wide Web. HTTPS is the secure variant through the use of Transport Layer Security protocols. HTTP is an application layer protocol.

Html

HTML stands for “HyperText Markup Language” which can be translated as “hypertext markup language”. It is used to create and represent the content of a web page and its structure.

Router

A router is a computer network equipment ensuring the routing of packets. Its role is to forward packets from one network interface to another, at best, according to a set of rules.
